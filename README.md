# Copy-playlist-from-adult-sites
Copies the playlist from many adult cam sites to the clipboard so you can play them in an external player or recorder

this userscript gets the HLS playlist from :

camsoda.com
streamray.com
cams.com (same site as streamray)
streamate.com (even while you need to log-in to see the link will work)
cam4.com
bongacams.com
flirt4free.com
xlovecam.com
myfreecams.com
chaturbate.com
olecams.com
dreamcam.com (2D stream)
eplay.com
and
sinparty.com (only the first cams are their own , other cams are streamate)

and show it in a pop-up 
This pop-up gives you the option to copy it to the clipboard

once it's on the clipboard you can open it in an external player like VLC

The scipt does use the sites own API's

And it's a user script so you need a userscript manager. 
This script is made and tested with Tampermonkey and firefox
